# BIN-BING-BINGO

A fraction bingo game built with React and TypeScript.

## Features

- Interactive fraction matching game
- Multiple difficulty levels
- Responsive design for all screen sizes
- Visual feedback for correct and incorrect answers
- Timer-based gameplay

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
